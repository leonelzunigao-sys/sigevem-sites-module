# SIGEVEM - Módulo de Administración de Sites

**Proyecto Terminal 2 - UnADM**  
**Desarrollado para:** H. Ayuntamiento de Ecatepec de Morelos

## 📖 Descripción
Este repositorio contiene el código fuente del módulo de **Administración de Sitios y Dispositivos Tecnológicos**. 

Este módulo fue solicitado por el cliente durante la ejecución del Proyecto Terminal 1 (PT1) debido a una necesidad operativa urgente. Se ha desarrollado manteniendo la arquitectura y estándares del sistema principal (SIGEVEM),
y servirá como el núcleo de trabajo para la finalización y entrega oficial en el **Proyecto Terminal 2 (PT2)**.

## ️ Tecnologías
- **Backend:** PHP 8.2 (Patrón MVC)
- **Base de Datos:** PostgreSQL + PostGIS
- **Frontend:** HTML5, CSS3, JavaScript, Leaflet.js

##  Estructura
- `config/`: Configuración de conexión y constantes.
- `modules/`: Lógica del módulo de sitios.
- `sql/`: Scripts de base de datos.
- `assets/`: Recursos estáticos.

## ⚠️ Nota
Este código corresponde al avance solicitado por el cliente para PT2. El sistema principal (PT1) se encuentra en otro repositorio/instancia.
